# Lightning

## ON DEVELOPEMENT
A language:
- Fast, reliable, scales by level.
- Python-like + pseudo elegance.
- Compatible with C
- LLVM support will coming soon after first release, also C++, Fortran compatibility

## LONG TERM GOALS
A language project which focus on elegance, readability and convenience.
It promises to bring power to users, scales by level, depth and need.
A personalized engine runs inside your device.
